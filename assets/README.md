# Assets Directory

This directory contains all static assets for the Online Pharmacy application.

## Structure

- **images/**: Contains all image files
  - **products/**: Product images (placeholder.png is provided as default)
  - **icons/**: SVG icons and other small graphics
  - **logo.png**: Application logo (to be added)
  - **favicon.ico**: Browser favicon (to be added)

- **fonts/**: Custom fonts (optional)

## Adding Assets

### Product Images
Place product images in `images/products/` directory. Update the `image` field in `data/products.json` to reference your images.

### Logo and Favicon
1. Add your logo as `images/logo.png`
2. Add your favicon as `images/favicon.ico`

### Icons
Place SVG icons in `images/icons/` directory for use throughout the application.
